# Friction Model Analysis

## 1. 论文模型回顾

### 1.1 基础自适应协同 (Adaptive Synergies, Grioli et al. 2012)

单个驱动器通过腱绳带动所有关节：

```
R q = x        (传动关系)
τ = R^T τ_M   (关节力矩)
```

其中 `R` 为传动矩阵，第 i 行对应第 i 根腱绳，第 j 列对应第 j 个关节。
R[i,j] = Σr_k (第 i 根腱绳上所有与第 j 个关节关联的滑轮半径之和)。

在外力和弹性力平衡下：

```
J^T f_ext + R^T τ_M = E q
```

解得关节角：

```
q = S σ + C J^T f_ext
```

其中 `S = E^{-1} R^T (R E^{-1} R^T)^{-1}` 为协同矩阵，`C` 为柔顺矩阵。

### 1.2 增强自适应协同 (Augmented Adaptive Synergies, Della Santina et al. 2018)

引入摩擦效应：腱绳在滑轮处有张力损失，形成第二个驱动方向 σ_f。

**原论文公式 (13-16)**：

沿路径的张力平衡：

```
T_j = T_{j-1} - V_j(v_j)    (V_j = 摩擦力)
```

速度平衡：

```
v_j = v_{j-1} + Σ r_{j,i} q̇_i
```

边界条件：

```
ṡ = -(v_0 + v_m)/2    (s = 滑动量)
τ_M = T_0 + T_m        (总驱动力)
```

矩阵形式 (13)：

```
M T + V(v) + e τ_M = 0
M v - R̄ q̇ = -2e ṡ
τ = - R̄^T T
```

其中 M 为 (m+1)×(m+1) 矩阵：

```
M = [[-1, 1, 0, ..., 0],
     [0, -1, 1, ..., 0],
     ...
     [1, 0, 0, ..., 1]]
```

**关键推导**：

由 (14)：`v = M^{-1} R̄ q̇ - 2e_v ṡ`，其中 `e_v = -M^{-1}e = [1,1,...,1]^T`

由 (15)：`τ = R^T τ_M + D(q̇, ṡ)`

其中：
- `R^T = - R̄^T e_v` — 基础传动矩阵
- `D(q̇, ṡ) = - R̄^T (-M^{-1} V(M^{-1} R̄ q̇ - 2e_v ṡ))` — 摩擦驱动项

**线性化 (17-20)**：

假设库仑摩擦模型 `V(v) = V_max tanh(v)`，在 q̇=0 时：

```
D(ṡ) = -R̄^T M^{-1} V_max tanh(2 e_v ṡ)
```

记 `u_2 = tanh(2 ṡ)`，得：

```
J^T f_ext = R^T u_1 + R_f^T u_2 - E q
```

其中：
- `R^T = - R̄^T e_v` — 第一驱动方向
- `R_f^T = - R̄^T M^{-1} V_max e_v` — 第二驱动方向（摩擦驱动）

**扩展传动矩阵 (22)**：

```
R_aug = [R; R_f]    (形状 2×n)
S_aug = R_aug⁺_E   (协同矩阵，形状 n×2)
```

**关键问题**：对于对称路径，`M^{-1} V_max e_v` 使得 `R_f = 0`。

### 1.3 论文的 Advanced Friction Model (Section V-A)

论文使用更精确的静摩擦模型 (Hayward & Armstrong 2000)：

```
z⁺_j = { θ_j + Δ_max_j  if θ_j ≤ z_j - Δ_max_j
       { θ_j - Δ_max_j  if θ_j ≥ z_j + Δ_max_j
       { z_j             otherwise
```

其中 z_j 为虚拟角度，Δ_max_j 为摩擦范围。摩擦力 = (θ_j - z_j) κ_j。

合并粘滞摩擦后，张力平衡：

```
T_j = T_{j-1} - v_j c_j / r²_j - (θ_j - z_j) κ_j / r²_j
```

矩阵形式 (39)：

```
M T + Λ v + Σ(θ - z) + e τ_M = 0
```

其中 Λ 和 Σ 为粘滞/静摩擦对角矩阵。

**关键物理**：该模型能产生非零 σ_f 效应，因为：
1. **粘滞摩擦**：`D(q̇, ṡ) = R^T M^{-1} Λ (M^{-1} R q̇ - e_v ṡ)` — 与速度相关
2. **静摩擦**：`D(θ,z) = R^T M^{-1} Σ(N M^{-1} R q - z) · e_v` — 与位移相关

通过 (41)：`θ = N(M^{-1} R q - e_v s)`，其中 N = diag(1/r_i)，θ 与 s 和 q 都相关。

**结论**：论文的完整摩擦模型当然可以产生 σ_f 效应，但其假设了整个腱绳路径上的**张力连续性**和**摩擦力平衡**——这意味着对于对称路径，如果所有滑轮半径相同，R_f 确实为 0。真正的非零 σ_f 效应来自于**张力的不对称分布**和**非线性的静摩擦记忆效应**。

## 2. 代码实现中的物理问题

### 2.1 旧模型 (`compute_R` 和 `compute_Rf`)

**`compute_R` (旧)**:
- 简单求和：`R[i,j] = Σr_k`，对所有在关节 j 上的滑轮 k
- 所有关节的传动比与滑轮数量成正比
- **问题**：中间关节与两端关节等同看待，忽略了 Capstan 张力衰减

**`compute_Rf` (旧)**:
- 实现论文公式 (20)：`R_f^T = - R̄^T M^{-1} V_max e_v`
- 用 `M^{-1} @ V_max @ e_v` 计算
- **问题**：对称路径上 Rf=0，且当 Rf 非零时与 R 不成比例时，
  两个 R 和 Rf 的尺度差异导致 `pinv` 将 Rf 的主导方向"投影"到 R 方向，
  使 σ_f 方向看起来像是 σ 方向。

### 2.2 真实物理行为

SoftHand 2 的物理行为（从论文 Fig. 9 和实验验证）：

**σ 模式（双电机同向）**：
- 张力从两端向中间 Capstan 指数衰减
- 中间滑轮有效张力减小
- 中间关节的等效传动比小于两端
- **U-shape**：拇指/小指 > 食指/无名指 > 中指

**σ_f 模式（双电机异向）**：
- Motor A 拉紧 → Motor A 附近滑轮张力高
- Motor B 放松 → Motor B 附近滑轮张力低
- 张力沿路径 Capstan 指数衰减
- 当张力衰减至静摩擦阈值以下 → 该段被"冻结"
- **手指重配**：A 侧手指闭合、B 侧打开（或反之）

## 3. Capstan + Slack 模型实现

这是我们的新模型，将论文的物理以更直接的方式实现：

### 3.1 σ 模式：Capstan 加权的 R

```
R[j] = Σ r_k · (exp(-β·d_Ak) + exp(-β·d_Bk)) / 2
```

其中：
- d_Ak = 元素 k 到 Motor A 的路径步数
- d_Bk = 元素 k 到 Motor B 的路径步数
- β = Capstan 摩擦系数（默认 0.09）

物理：张力从两端向中间衰减，中间滑轮有效传动比小。

### 3.2 σ_f 模式：张力差加权的 Rf

```
Rf[j] = Σ r_k · (exp(-β·d_Ak) - exp(-β·d_Bk))
```

物理：
- Motor A 拉紧时，A 侧贡献 d_Ak 近的权重
- Motor B 拉紧时，B 侧贡献 d_Bk 近的权重
- 差产生不对称 → 手指重配
- 当两边的张力衰减都低于静摩擦阈值 → dead zone，贡献为 0

## 5. 关键物理见解

1. **Capstan 衰减**是摩擦驱动的核心机制，不引入此效应则无法解释
   σ 模式的 U-shape 和 σ_f 模式的 Dead Zone。

2. **腱绳单向传力**（不能受推）导致 slack 侧张力为 0，
   而非对称分布，这放大了 σ_f 模式的不对称性。

3. **对称路径的 Rf=0** 是库仑线性化模型的数学结果，在真实静摩擦模型下
   会产生非零效应（通过记忆效应），但在实际控制中可通过人为引入
   非对称性（如不同半径的滑轮、不同材料的摩擦系数）来获得更大的 σ_f 效果。

4. **SVD 稳定性**：当 R 和 Rf 满秩时（非对称路径），
   AugmentedAdaptiveSynergyModel 可正确分解出两个独立的协同方向。
   当 Rf=0 时（对称路径），rank-deficient 检测自动禁用 σ_f 方向。
